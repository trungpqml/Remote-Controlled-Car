Camera Stream
1. run command in terminal      "python camera.py"
2. open a web browser in a laptop
3. enter the IP address with port 8000

Motor Control

1. run command in terminal      "python motor.py"
2. enter proper keys to control the motor

Key list
w = run forward
s = run backward
a = turn left
d = turn right
x = stop
j = low speed
k = medium speed
l = high speed
